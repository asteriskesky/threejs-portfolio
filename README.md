# Akash Ojha - 3D Portfolio

An interactive 3D portfolio website built with **React Three Fiber** and **Three.js**. This project features a cinematic 3D workspace with a playable MacBook model that hosts the main portfolio content, bridging the gap between immersive 3D web experiences and standard web accessibility.

## âœ¨ Features

*   **Interactive 3D Scene**: A fully navigable 3D workspace built with React Three Fiber.
*   **Embedded HTML**: Seamless integration of standard HTML/CSS content within the 3D MacBook screen using `drei/Html`.
*   **Custom Animations**: Smooth camera transitions, laptop opening sequences, and interactive lighting.
*   **Performance Optimized**: Built with Vite for fast HMR and optimized production builds.
*   **Responsive**: Designed to work across different screen sizes.
